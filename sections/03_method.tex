\section{Methodology}
\label{sec:method}

\subsection{Preliminaries and Internal Activations}

We build on the stochastic-interpolant formulation used by SiT~\citep{ma2024sit}. Given a data latent $x_0\sim p_{\mathrm{data}}$, Gaussian noise $\epsilon\sim\mathcal{N}(0,I)$, and timestep $t\in[0,1]$, the interpolation path is
\begin{equation}
x_t=\alpha_t x_0+\sigma_t\epsilon ,
\label{eq:stochastic_interpolant}
\end{equation}
with the target velocity
\[
u_t=\dot{\alpha}_t x_0+\dot{\sigma}_t\epsilon .
\]
A diffusion transformer $v_\theta$ predicts this velocity from $(x_t,t,c)$ by minimizing
\begin{equation}
\mathcal{L}_{\mathrm{FM}}(\theta)
=
\mathbb{E}_{x_0,\epsilon,t,c}
\left[
\left\|v_\theta(x_t,t,c)-u_t\right\|_2^2
\right].
\label{eq:flow_matching_loss}
\end{equation}

\paragraph{Motivation from shared structure.}
SRA observes that diffusion-transformer representations become more structured across depth and denoising time~\citep{jiang2025sra}. We follow this observation in \Cref{fig:projection_layout_observation}: internal layers differ in semantic level, but often preserve a common spatial scaffold across timesteps and block depth. Directly matching a fixed pair of raw activations ties supervision to one representation gap. LARA instead learns a shared transition interface: common structure is captured by shared parameters, while layer embeddings adapt the mapping to each source--target pair.

\subsection{Layer Aware Representation Alignment}

\paragraph{Layer-aware transition prediction.}

Let $A_{\theta,\ell}^t\in\mathbb{R}^{N\times C}$ denote the token activation produced by the diffusion transformer after block $\ell$ when processing $(x_t,t,c)$, where $N$ is the number of latent tokens and $C$ is the channel dimension. We write $F_{\theta,b:L}$ for the remaining model suffix that consumes a depth-$b$ activation and produces the final velocity prediction; this is the ordinary transformer suffix after the insertion point, not a separate network. For readability, we omit $\theta$ from $A_{\theta,\ell}^t$ when the dependency is clear.

Given a source layer $a$ and target layer $b$ with $a<b$, LARA predicts the target-layer activation tokens from the source activation:
\begin{equation}
\widehat A_{a\to b}^{\,t}
=
G_\psi(A_a^t,e_a,e_b,e_t),
\qquad
\widehat A_{a\to b}^{\,t}\in\mathbb{R}^{N\times C}.
\label{eq:realign_predictor}
\end{equation}
Here $G_\psi$ is a lightweight transition predictor, and $e_a,e_b,e_t$ are source-layer, target-layer, and timestep embeddings. The predicted activation $\widehat A_{a\to b}^{\,t}$ lives in the coordinate system of layer $b$. The same predictor is reused for all valid layer pairs, which removes the need to train a separate adapter for each manually selected pair.

The transition prediction loss is
\begin{equation}
\mathcal{L}_{\mathrm{pred}}(\theta,\psi)
=
\mathbb{E}_{a<b,t}
\left[
d\left(
\widehat A_{a\to b}^{\,t},
\operatorname{sg}(A_b^t)
\right)
\right],
\label{eq:align_loss}
\end{equation}
where $d(\cdot,\cdot)$ is a tokenwise cosine distance and $\operatorname{sg}$ stops gradients through the target activation. In implementation, the predictor output is formed before token normalization, as detailed in \Cref{alg:lara_transition_distance}. This loss updates the predictor and the source-side backbone representation, while the target layer serves as a stable internal reference.

\textbf{Shared predictor training.}
At each iteration, LARA samples source--target layer pairs and trains one shared predictor with $\mathcal{L}_{\mathrm{pred}}$. This passes supervision across depth by predicting a target-layer state in layer $b$'s coordinate system, rather than directly matching raw activations from two semantically different depths. The resulting signal improves training dynamics over the baseline and LayerSync, as shown in \Cref{fig:training_curve}.

\paragraph{Transition consistency.}

A single long transition can be difficult to learn directly. To propagate supervision across distant layers without committing to one fixed pair, LARA encourages direct transitions to agree with compositions of shorter transitions. We keep an EMA copy $G_{\bar\psi}$ of the predictor for the composed target, while the intermediate $a\to b$ state is the cached online prediction from \Cref{eq:align_loss}. Thus the direct branch uses the online predictor, and the composed branch provides a stop-gradient EMA target. For $a<b<c$, we define

\begin{equation}
\widehat A_{a\to c}^{\,t}
=
G_\psi(\operatorname{sg}(A_a^t),e_a,e_c,e_t),
\qquad
\widetilde A_{a\to b\to c}^{\,t}
=
G_{\bar\psi}\!\left(
\operatorname{sg}(\widehat A_{a\to b}^{\,t}),
e_b,e_c,e_t
\right),
\label{eq:direct_transition}
\end{equation}
The consistency loss is
\begin{equation}
\mathcal{L}_{\mathrm{cons}}(\psi;\bar\psi)
=
\mathbb{E}_{a<b<c,t}
\left[
d\left(
\widehat A_{a\to c}^{\,t},
\operatorname{sg}(\widetilde A_{a\to b\to c}^{\,t})
\right)
\right].
\label{eq:consistency_loss}
\end{equation}
Only the online predictor is updated by this term. The source activation and composed target are detached, so the constraint regularizes the transition interface rather than forcing the backbone to collapse intermediate layers.

\noindent
\begin{minipage}[t]{0.57\linewidth}
\textbf{Diversity regularization.} LARA also includes a lightweight diversity term inspired by DiverseDiT~\citep{yang2026diversedit}. The transition objective deliberately increases the shared signal between depths: a source layer should contain enough information to predict a later target layer. If this sharing is left unconstrained, however, adjacent hidden states can become overly dependent and carry the same features. We therefore reduce redundant cross-layer dependence by penalizing squared cosine similarity. This is a tractable proxy for lowering excessive mutual information between layer activations, without requiring explicit mutual-information estimation. For randomly sampled layer pairs $(i,j)$, independent of the transition pair used above, we define
\begin{equation}
\mathcal{L}_{\mathrm{div}}(\theta)
=
\mathbb{E}_{i<j,t}
\left[
\operatorname{cos}^2(A_i^t,A_j^t)
\right].
\label{eq:diversity_loss}
\end{equation}
Minimizing this term discourages layer homogenization: different depths can still share the structure needed for transition prediction, but they are pushed away from encoding identical directions. The heatmap diagnostic in \Cref{fig:cosine_squared_heatmap} visualizes this intended effect as lower pairwise layer similarity after applying the regularizer.
\end{minipage}

\paragraph{Full objective.}

The LARA regularizer assigns a separate weight to each component:
\begin{equation}
\mathcal{L}_{\mathrm{LARA}}(\theta,\psi)
=
\lambda_{\mathrm{pred}}\mathcal{L}_{\mathrm{pred}}(\theta,\psi)
+
\lambda_{\mathrm{cons}}\mathcal{L}_{\mathrm{cons}}(\psi;\bar\psi)
+
\lambda_{\mathrm{div}}\mathcal{L}_{\mathrm{div}}(\theta).
\label{eq:realign_loss}
\end{equation}
The final training objective: 
\begin{equation}
\mathcal{L}(\theta,\psi)
=
\mathcal{L}_{\mathrm{FM}}(\theta)
+
\mathcal{L}_{\mathrm{LARA}}(\theta,\psi).
\label{eq:total_loss}
\end{equation}
The update paths are explicit: $\mathcal{L}_{\mathrm{pred}}$ updates the predictor and source-side backbone, $\mathcal{L}_{\mathrm{cons}}$ updates only the predictor, and $\mathcal{L}_{\mathrm{div}}$ updates only the DiT. Training procedures, predictor capacity, layer-pair sampling, and loss weights are reported in \Cref{app:realign_algorithm_compute,app:realign_hyperparameters}.
