\section{Method}
\label{sec:method}

\subsection{Preliminaries}

We adopt the stochastic-interpolant formulation used by SiT~\citep{ma2024sit}, which provides a common view of flow-based and diffusion-based generative models. Given a data latent $x_0\sim p_{\mathrm{data}}$, Gaussian noise $\epsilon\sim\mathcal{N}(0,I)$, and timestep $t\in[0,1]$, the interpolation path is
\begin{equation}
x_t=\alpha_t x_0+\sigma_t\epsilon ,
\label{eq:stochastic_interpolant}
\end{equation}
where $\alpha_t$ and $\sigma_t$ control the mixture of data and noise. The schedule is chosen so that the path connects clean data to a tractable noise distribution, and generation follows the learned dynamics in the reverse direction. The instantaneous velocity of this path is
\[
u_t=\dot{\alpha}_t x_0+\dot{\sigma}_t\epsilon .
\]
A diffusion transformer $v_\zeta$ receives the noisy latent $x_t$, timestep $t$, and optional condition $c$, and is trained to predict this velocity by minimizing
\begin{equation}
\mathcal{L}_{\mathrm{FM}}
=
\mathbb{E}_{x_0,\epsilon,t,c}\left[
\left\|v_\zeta(x_t,t,c)-u_t\right\|_2^2
\right].
\label{eq:flow_matching_loss}
\end{equation}

\subsection{From Projection Interfaces to Self-Alignment}

\paragraph{Representation hierarchy.}
Diffusion transformers do not expose a uniform stack of interchangeable hidden states. Prior analyses show that intermediate layers differ in semantic quality, with useful features often peaking before the final decoder-like blocks~\citep{mukhopadhyay2024,kim2025,stracke2025,haghighi2026layersync}. We observe the same pattern in a pretrained SiT-XL/2: activation scale drifts across depth, while CKA to DINOv2 varies substantially by layer (\Cref{fig:sit_xl_layer_statistics}). The first effect reflects a statistical mismatch between raw layer states, consistent with signal-propagation and Transformer normalization analyses~\citep{glorot2010understanding,he2015delving,schoenholz2017deep,liu2020understandingtransformers}; the second reflects a representation hierarchy.

\paragraph{Projection as interface.}
LayerSync exploits the hierarchy by aligning weaker layers to richer internal ones~\citep{haghighi2026layersync}, but direct alignment also assumes that source and target activations are comparable in raw coordinates. This makes layer choice a coupled problem: a poor source layer may be semantically weak, statistically mismatched, or both. REPA and SRA avoid raw matching by inserting projection heads before matching external or internal targets~\citep{yu2024repa,jiang2025sra}. We isolate this design choice in \Cref{tab:layersync_projector_probe}.

\paragraph{Signal filtering.}
The weak-layer rows are the key comparison. If projection improves them, the source is not simply uninformative; its useful signal is poorly exposed in raw coordinates. In a transformer architecture with residual blocks, $A_b^t=A_a^t+\Delta_{a:b}^t$, so a projector satisfying $P_{a\to b}(A_a^t)\approx A_b^t$ learns the recoverable part of this accumulated update in the target layer's coordinates. The interface can therefore extract useful supervision from a source activation, distill information between representation spaces, and filter the auxiliary signal before it updates the flow-matching model. We test this view in the appendix with gradient cosine between $\mathcal{L}_{\mathrm{FM}}$ and direct versus projected alignment losses.

\paragraph{Recoverable layout.}
The recoverable signal may include spatial structure, not only global semantics. iREPA shows that preserving spatial structure in REPA targets can matter beyond global classification strength~\citep{singh2025irepa}. We therefore use PCA and stable rank in \Cref{fig:projection_layout_observation} as a compact qualitative check: low stable rank means a few principal directions explain much of the variation, making PCA maps informative about coarse layout~\citep{daneshmand2020batchnormrankcollapse,feng2022rankdiminishing}. We treat layout recovery as a hypothesis rather than a primary claim.

\subsection{ReSync: Self-Aligned Transition Consistency}

ReSync turns the projection-interface view into an amortized self-alignment objective. Instead of selecting a small set of teacher layers or training a separate adapter for each pair, it samples transitions across the valid layer range so that useful signals from many internal states can supervise many source layers. Let $A_\ell^t\in\mathbb{R}^{N\times C}$ denote the token activation after block $\ell$ when the diffusion transformer processes $(x_t,t,c)$. For a source layer $a$ and a later target layer $b$, ReSync predicts the target representation from the source representation at the same denoising timestep:
\begin{equation}
T_{a\to b}^t(A)\equiv T_\psi(A,a,b,t),
\qquad
\hat A_{b\mid a}^t=T_{a\to b}^t(A_a^t).
\label{eq:resync_predictor}
\end{equation}
The predictor is shared across pairs and conditioned on $(a,b,t)$, making the interface amortized rather than pair-specific. It learns an implicit internal mapping from $A_a^t$ to the target-layer coordinates of $A_b^t$. Unless stated otherwise, $\mathbb{E}$ averages over data, noise, timesteps, and sampled layer indices; $d$ is tokenwise cosine distance; and $\mathrm{sg}$ denotes stop-gradient. The self-alignment loss is
\begin{equation}
\mathcal{L}_{\mathrm{pred}}
=
\mathbb{E}\left[
d\left(\hat A_{b\mid a}^t,\mathrm{sg}(A_b^t)\right)
\right].
\label{eq:align_loss}
\end{equation}
The target activation is detached, so supervision flows through the transition interface while avoiding symmetric collapse between source and target states.

An amortized predictor trained only with pairwise targets can still behave like a set of unrelated adapters. To couple its predictions into a single internal transition rule, ReSync imposes path consistency. For a triplet $a<b<c$, the direct transition should agree with the composition of two shorter transitions:
\begin{equation}
\hat A_{c\mid a}^t
\approx
\hat A_{c\mid b\mid a}^t .
\label{eq:transition_composition}
\end{equation}
Here $\hat A_{c\mid a}^t$ is produced by one application of $T_\psi$, while $\hat A_{c\mid b\mid a}^t$ is obtained by applying the transitions $a\to b$ and $b\to c$. We form the composed target with an EMA copy $\bar T$ of the predictor, writing $\bar T_{p\to q}^t(A)\equiv\bar T(A,p,q,t)$ and following the common use of weight-averaged consistency targets~\citep{tarvainen2017meanteacher}:
\begin{equation}
\tilde A_a^t=\mathrm{sg}(A_a^t),
\qquad
\bar A_{b\mid a}^t=\bar T_{a\to b}^{\,t}(\tilde A_a^t),
\qquad
\bar A_{c\mid b\mid a}^t
=
\bar T_{b\to c}^{\,t}(\bar A_{b\mid a}^t).
\label{eq:composed_target}
\end{equation}
The online predictor is then trained against this composed target:
\begin{equation}
\mathcal{L}_{\mathrm{cons}}
=
\mathbb{E}\left[
d\left(T_{a\to c}^t(\tilde A_a^t),\mathrm{sg}(\bar A_{c\mid b\mid a}^t)\right)
\right].
\label{eq:consistency_loss}
\end{equation}
This constraint is the main distinction from pairwise self-alignment. The same conditional predictor must explain local and longer-range transitions sampled throughout the model, making ReSync a consistent interface for reusing internal guidance rather than a collection of fixed-pair matching losses. Implementation choices for layer ranges, gap sampling, and predictor capacity are given in \Cref{app:transition_pair_predictor_design}.

State-level prediction can still be satisfied by activations that are locally close to the target but brittle under the remaining backbone. We therefore require predicted activations to remain compatible with downstream computation. Let $F_{\zeta,b:L}$ denote the suffix of the diffusion transformer from block $b$ to the output. Given a predicted target activation, we compute
\begin{equation}
o_{\mathrm{pred}}
=
F_{\zeta,b:L}(\hat A_b^t,t,c),
\qquad
o_{\mathrm{full}}
=
F_{\zeta,b:L}(A_b^t,t,c),
\label{eq:output_consistency_forward}
\end{equation}
and enforce output consistency with the full-state branch:
\begin{equation}
\mathcal{L}_{\mathrm{out}}
=
\left\|
o_{\mathrm{pred}}
-
\mathrm{sg}(o_{\mathrm{full}})
\right\|_2^2 .
\label{eq:output_consistency_loss}
\end{equation}
This term is not an external teacher loss: both branches use the same backbone suffix, and the full-state branch is detached. It penalizes predictions that match the target activation locally but drift away from the representation manifold needed by the remaining blocks, making the transition predictor more robust to its own errors.

To preserve functional separation between blocks, we add a lightweight diversity term inspired by DiverseDiT~\citep{yang2026diversedit}. For randomly sampled layer pairs $(i,j)$, independent of the transition pair used in $\mathcal{L}_{\mathrm{pred}}$, we use
\begin{equation}
\mathcal{L}_{\mathrm{div}}
=
\mathbb{E}_{i<j,t}
\left[
\operatorname{cos}^2(A_i^t,A_j^t)
\right].
\label{eq:diversity_loss}
\end{equation}
It penalizes high direct similarity between sampled layer activations while leaving the transition targets unchanged; no-diversity ablations isolate its effect.

\paragraph{Gradient routing.}
We route gradients according to the role of each term. $\mathcal{L}_{\mathrm{pred}}$ updates both the transition predictor and the source-side backbone activations, while the target activation is stopped. $\mathcal{L}_{\mathrm{cons}}$ updates only the online predictor because both the source activation and the EMA-composed target are detached. $\mathcal{L}_{\mathrm{out}}$ updates the predictor and the active backbone suffix through the predicted branch, while the full-state branch is stopped. $\mathcal{L}_{\mathrm{div}}$ is applied only to backbone activations and does not update the transition predictor.

\subsection{Objective}

ReSync is added to the flow-matching objective as an internal regularizer. It combines transition prediction, path consistency, output consistency, and weak layer diversity:
\begin{equation}
\mathcal{L}_{\mathrm{ReSync}}
=
\mathcal{L}_{\mathrm{pred}}
+
\beta_{\mathrm{cons}}\mathcal{L}_{\mathrm{cons}}
+
\beta_{\mathrm{out}}\mathcal{L}_{\mathrm{out}}
+
\beta_{\mathrm{div}}\mathcal{L}_{\mathrm{div}} .
\label{eq:resync_loss}
\ref{}\end{equation}
The final training objective is
\begin{equation}
\mathcal{L}
=
\mathcal{L}_{\mathrm{FM}}
+
\lambda_{\mathrm{sync}}\mathcal{L}_{\mathrm{ReSync}} .
\label{eq:total_loss}
\end{equation}
The predictor is used only during training and is discarded during standard sampling, so ReSync adds no inference-time cost. Training and inference procedures, architecture, parameter overhead, and loss weights are reported in \Cref{app:resync_algorithm_compute,app:resync_hyperparameters}; inference-time skipped-depth execution is a possible extension evaluated separately in \Cref{sec:skip_prediction}.
