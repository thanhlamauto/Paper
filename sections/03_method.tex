\section{Method}
\label{sec:method}

\subsection{Preliminaries and Internal Activations}

We build on the stochastic-interpolant formulation used by SiT~\citep{ma2024sit}. Given a data latent $x_0\sim p_{\mathrm{data}}$, Gaussian noise $\epsilon\sim\mathcal{N}(0,I)$, and timestep $t\in[0,1]$, the interpolation path is
\begin{equation}
x_t=\alpha_t x_0+\sigma_t\epsilon ,
\label{eq:stochastic_interpolant}
\end{equation}
with target velocity
\[
u_t=\dot{\alpha}_t x_0+\dot{\sigma}_t\epsilon .
\]
A diffusion transformer $v_\theta$ predicts this velocity from $(x_t,t,c)$ by minimizing
\begin{equation}
\mathcal{L}_{\mathrm{FM}}(\theta)
=
\mathbb{E}_{x_0,\epsilon,t,c}
\left[
\left\|v_\theta(x_t,t,c)-u_t\right\|_2^2
\right].
\label{eq:flow_matching_loss}
\end{equation}

Let $A_{\theta,\ell}^t\in\mathbb{R}^{N\times C}$ denote the token activation produced by the diffusion transformer after block $\ell$ when processing $(x_t,t,c)$, where $N$ is the number of latent tokens and $C$ is the channel dimension. We write $F_{\theta,b:L}$ for the remaining model suffix that consumes a depth-$b$ activation and produces the final velocity prediction; this is the ordinary transformer suffix after the insertion point, not a separate network. For readability, we omit $\theta$ from $A_{\theta,\ell}^t$ when the dependency is clear.

\paragraph{Motivation from shared structure.}
SRA observes that diffusion-transformer representations become more structured across depth and denoising time~\citep{jiang2025sra}. We follow this observation in \Cref{fig:projection_layout_observation}: internal layers differ in semantic level, but often preserve a common spatial scaffold. Directly matching a fixed pair of raw activations ties supervision to one representation gap. LARA instead learns a shared transition interface: common structure is captured by shared parameters, while layer embeddings adapt the mapping to each source--target pair.

\subsection{Layer-Aware Transition Prediction}

Given a source layer $a$ and target layer $b$ with $a<b$, LARA predicts the target-layer activation tokens from the source activation:
\begin{equation}
\widehat A_{a\to b}^{\,t}
=
G_\psi(A_a^t,e_a,e_b,e_t),
\qquad
\widehat A_{a\to b}^{\,t}\in\mathbb{R}^{N\times C}.
\label{eq:realign_predictor}
\end{equation}
Here $G_\psi$ is a lightweight transition predictor, and $e_a,e_b,e_t$ are source-layer, target-layer, and timestep embeddings. The predicted activation $\widehat A_{a\to b}^{\,t}$ lives in the coordinate system of layer $b$. The same predictor is reused for all valid layer pairs, which removes the need to train a separate adapter for each manually selected pair.

The transition prediction loss is
\begin{equation}
\mathcal{L}_{\mathrm{pred}}(\theta,\psi)
=
\mathbb{E}_{a<b,t}
\left[
d\left(
\widehat A_{a\to b}^{\,t},
\operatorname{sg}(A_b^t)
\right)
\right],
\label{eq:align_loss}
\end{equation}
where $d(\cdot,\cdot)$ is tokenwise cosine distance and $\operatorname{sg}$ stops gradients through the target activation. This loss updates the predictor and the source-side backbone representation, while the target layer serves as a stable internal reference.

\subsection{Transition Consistency}

A single long transition can be difficult to learn directly. To propagate supervision across distant layers without committing to one fixed pair, LARA encourages direct transitions to agree with compositions of shorter transitions. For $a<b<c$, we define
\begin{equation}
\widehat A_{a\to c}^{\,t}
=
G_\psi(\operatorname{sg}(A_a^t),e_a,e_c,e_t),
\qquad
\widetilde A_{a\to b\to c}^{\,t}
=
G_{\mathrm{ema}}\!\left(
G_{\mathrm{ema}}(\operatorname{sg}(A_a^t),e_a,e_b,e_t),
e_b,e_c,e_t
\right),
\label{eq:direct_transition}
\end{equation}
where $G_{\mathrm{ema}}$ is an EMA copy of the predictor~\citep{tarvainen2017meanteacher}. The consistency loss is
\begin{equation}
\mathcal{L}_{\mathrm{cons}}(\psi)
=
\mathbb{E}_{a<b<c,t}
\left[
d\left(
\widehat A_{a\to c}^{\,t},
\operatorname{sg}(\widetilde A_{a\to b\to c}^{\,t})
\right)
\right].
\label{eq:consistency_loss}
\end{equation}
Only the online predictor is updated by this term. The source activation and the composed EMA target are detached, so the constraint regularizes the transition interface rather than forcing the backbone to collapse intermediate layers.

\subsection{Diversity Regularization}

We also use a lightweight diversity regularizer inspired by DiverseDiT~\citep{yang2026diversedit}. For randomly sampled layer pairs $(i,j)$, independent of the transition pair used above, we define
\begin{equation}
\mathcal{L}_{\mathrm{div}}(\theta)
=
\mathbb{E}_{i<j,t}
\left[
\operatorname{cos}^2(A_i^t,A_j^t)
\right].
\label{eq:diversity_loss}
\end{equation}
This term discourages excessive layer homogenization. It keeps different depths from carrying indistinguishable information, reducing ambiguity while preserving the shared structure that the transition predictor exploits.

\subsection{Full Objective}

The LARA regularizer assigns a separate weight to each component:
\begin{equation}
\mathcal{L}_{\mathrm{LARA}}(\theta,\psi)
=
\lambda_{\mathrm{pred}}\mathcal{L}_{\mathrm{pred}}(\theta,\psi)
+
\lambda_{\mathrm{cons}}\mathcal{L}_{\mathrm{cons}}(\psi)
+
\lambda_{\mathrm{div}}\mathcal{L}_{\mathrm{div}}(\theta).
\label{eq:realign_loss}
\end{equation}
The final training objective is
\begin{equation}
\mathcal{L}(\theta,\psi)
=
\mathcal{L}_{\mathrm{FM}}(\theta)
+
\mathcal{L}_{\mathrm{LARA}}(\theta,\psi).
\label{eq:total_loss}
\end{equation}
The update paths are explicit: $\mathcal{L}_{\mathrm{pred}}$ updates the predictor and source-side backbone, $\mathcal{L}_{\mathrm{cons}}$ updates only the predictor, and $\mathcal{L}_{\mathrm{div}}$ updates only the DiT. Training procedures, predictor capacity, layer-pair sampling, and loss weights are reported in \Cref{app:realign_algorithm_compute,app:realign_hyperparameters}.
