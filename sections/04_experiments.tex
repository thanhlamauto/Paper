\section{Experiments}
\label{sec:experiments}

This section evaluates ReSync as a mechanistic hypothesis: if internal guidance is more useful when it models representation evolution across depth, then transition prediction and compositional consistency should improve generation while producing measurable gains in transition-level diagnostics. We organize the experiments around three questions:
\begin{itemize}[leftmargin=*]
    \item Does ReSync improve generation quality across domains under matched budgets?
    \item How do ReSync representations and transition diagnostics compare with internal-alignment baselines?
    \item Can the learned predictor be used for exploratory skipped-depth execution?
\end{itemize}

\subsection{Image Generation}
\label{sec:image_generation}
\label{sec:experimental_setup}

\paragraph{Implementation details.}
Unless otherwise stated, we follow the class-conditional ImageNet training setup of SiT~\citep{ma2024sit}. We train on ImageNet~\citep{deng2009imagenet} at $256\times256$ resolution using the preprocessing protocol of ADM~\citep{dhariwal2021diffusion}. Each image is encoded by the Stable Diffusion VAE from LDM~\citep{rombach2022ldm} into a latent tensor $z\in\mathbb{R}^{32\times32\times4}$. We evaluate the \texttt{B/2}, \texttt{L/2}, and \texttt{XL/2} SiT configurations with patch size $2$. ReSync trains the transition predictor jointly with the backbone using \Cref{eq:total_loss}. Unless specified, we use AdamW with learning rate $10^{-4}$ and batch size $256$. Additional details on architecture, transition-pair sampling, EMA decay, loss weights, training length, hardware, and compute budget are organized in \Cref{app:experimental_details}.

\paragraph{Evaluation.}
We report Frechet Inception Distance (FID)~\citep{heusel2017gans}, Inception Score (IS)~\citep{salimans2016improved}, precision, and recall~\citep{kynkaanniemi2019precision} using $50\mathrm{K}$ generated samples. We use the same reference statistics and evaluation implementation for all ReSync variants and baselines. Definitions for the diagnostic metrics that are not standard generation benchmarks are provided in \Cref{app:metric_definitions}. \textbf{[PLACEHOLDER: specify the exact evaluator, reference statistics, classifier-free guidance setting, sampling steps, and whether sFID is reported in the main table or appendix]}.

\paragraph{Comparison methods.}
Our main comparisons are against the base SiT/DiT backbone and recent representation-regularized training methods. For self-contained guidance, we compare with Dispersive Loss~\citep{wang2025dispersive}, SRA~\citep{jiang2025sra}, and LayerSync~\citep{haghighi2026layersync}. For externally guided training, we compare with REPA~\citep{yu2024repa} and REED~\citep{wang2025flexible}. For completeness, we also report representative image-generation baselines: pixel-space diffusion models including ADM~\citep{dhariwal2021diffusion}, VDM++~\citep{kingma2023understanding}, Simple Diffusion~\citep{hoogeboom2023simple}, and CDM~\citep{ho2022cascaded}; latent diffusion and transformer-based models including LDM~\citep{rombach2022ldm}, U-ViT-H/2~\citep{bao2023all}, DiffiT~\citep{hatamizadeh2024diffit}, MDTv2-XL/2~\citep{gao2023mdtv2}, MaskDiT~\citep{zheng2023fast}, SD-DiT~\citep{zhu2024sddit}, DiT~\citep{peebles2023scalable}, and SiT~\citep{ma2024sit}; and autoregressive or predictive-generation baselines such as VAR~\citep{tian2024visual} and D-JEPA~\citep{chen2024jepa}. \textbf{[PLACEHOLDER: move detailed baseline descriptions and parameter matching protocol to appendix]}.

\subsection{Transition Diagnostics}
\label{sec:transition_diagnostics}

Generation metrics alone cannot show whether ReSync is using the intended mechanism. We therefore measure whether the learned predictor behaves like a reusable depth-transition model. The main diagnostics are transition prediction error on sampled layer pairs, held-out prediction error on unseen layer pairs, compositional error between direct and composed transitions, and the relationship between these errors and generation quality. These measurements do not prove a causal mechanism, but they test whether the empirical gains are consistent with the transition-learning interpretation.
