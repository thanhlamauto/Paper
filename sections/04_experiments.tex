\section{Experiments}
\label{sec:experiments}

This section evaluates ReSync as a mechanistic hypothesis: if internal guidance is more useful when it models representation evolution across depth, then transition prediction and compositional consistency should improve generation while producing measurable gains in transition-level diagnostics. Following this view, our experiments are organized around five questions:
\begin{itemize}[leftmargin=*]
    \item Does ReSync improve generation quality under matched training budgets? See \Cref{tab:comparison_external_guidance,fig:training_curves,tab:model_scaling}.
    \item Which ReSync components are responsible for the gain, including timestep conditioning of depth transitions and compositional consistency? See \Cref{tab:resync_ablation}.
    \item Does the transition predictor learn reusable internal dynamics? See \Cref{tab:transition_diagnostics}.
    \item Is the benefit different from adding a local projection head? See \Cref{tab:projection_transition_ablation}.
    \item As an exploratory ablation, can predicted hidden states support skipped-depth prediction? See \Cref{tab:skip_prediction}.
\end{itemize}

\subsection{Experimental Setup}
\label{sec:experimental_setup}

\paragraph{Implementation details.}
Unless otherwise stated, we follow the standard latent diffusion-transformer training recipe used by DiT and SiT-style baselines~\citep{ma2024sit}. Models are trained with AdamW using a constant learning rate of $10^{-4}$, no weight decay, and a batch size of $256$. Images are encoded into latent space using the Stable Diffusion VAE, and we evaluate the \texttt{B/2}, \texttt{L/2}, and \texttt{XL/2} model configurations with patch size $2$. For ReSync, the transition predictor is trained jointly with the backbone using the auxiliary objective in \Cref{eq:total_loss}. \subsection{Transition Diagnostics}
\label{sec:transition_diagnostics}

Generation metrics alone cannot show whether ReSync is using the intended mechanism. We therefore measure whether the learned predictor behaves like a reusable depth-transition model. The main diagnostics are transition prediction error on sampled layer pairs, held-out prediction error on unseen layer pairs, compositional error between direct and composed transitions, and the relationship between these errors and generation quality. We report cross-timestep prediction only as a preliminary diagnostic because it requires additional target activations. These measurements do not prove a causal mechanism, but they test whether the empirical gains are consistent with the transition-learning interpretation.

\subsection{Skipped-Depth Prediction}
\label{sec:skip_prediction}

Because ReSync predicts future hidden states, it provides a natural way to test whether the learned representation is useful beyond a hidden-state similarity loss. We evaluate skipped-depth prediction by replacing a block interval with the predicted hidden state and then resuming the remaining backbone. The primary measurements are output error against the full model, generation quality under skipped execution, and wall-clock speed. We treat this as an auxiliary capability rather than the central claim.
