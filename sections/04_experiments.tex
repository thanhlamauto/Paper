\section{Experiments}
\label{sec:experiments}

This section evaluates ReSync as a mechanistic hypothesis: if internal guidance is more useful when it models representation evolution across depth, then transition prediction and compositional consistency should improve generation while producing measurable gains in transition-level diagnostics. Following this view, our experiments are organized around five questions:
\begin{itemize}[leftmargin=*]
    \item Does ReSync improve generation quality under matched training budgets? See \Cref{tab:comparison_external_guidance,fig:training_curves,tab:model_scaling}.
    \item Which ReSync components are responsible for the gain, including timestep conditioning of depth transitions and compositional consistency? See \Cref{tab:resync_ablation}.
    \item Does the transition predictor learn reusable internal dynamics? See \Cref{tab:transition_diagnostics}.
    \item Does projection reduce raw layer mismatch rather than merely adding local capacity? See \Cref{fig:projection_mismatch_observation,fig:projection_layout_observation,tab:projection_transition_ablation}.
    \item As an exploratory ablation, can predicted hidden states support skipped-depth prediction? See \Cref{tab:skip_prediction}.
\end{itemize}

\subsection{Image Generation}
\label{sec:image_generation}
\label{sec:experimental_setup}

\paragraph{Implementation details.}
Unless otherwise stated, we follow the class-conditional ImageNet training setup of SiT~\citep{ma2024sit}. We train on ImageNet~\citep{deng2009imagenet} at $256\times256$ resolution using the preprocessing protocol of ADM~\citep{dhariwal2021diffusion}. Each image is encoded by the Stable Diffusion VAE from LDM~\citep{rombach2022ldm} into a latent tensor $z\in\mathbb{R}^{32\times32\times4}$. We evaluate the \texttt{B/2}, \texttt{L/2}, and \texttt{XL/2} SiT configurations with patch size $2$. ReSync trains the transition predictor jointly with the backbone using \Cref{eq:total_loss}. Unless specified, we use AdamW with learning rate $10^{-4}$ and batch size $256$. \paragraph{Comparison methods.}
Our main comparisons are against the base SiT/DiT backbone and recent representation-regularized training methods. \subsection{Transition Diagnostics}
\label{sec:transition_diagnostics}

Generation metrics alone cannot show whether ReSync is using the intended mechanism. We therefore measure whether the learned predictor behaves like a reusable depth-transition model. The main diagnostics are transition prediction error on sampled layer pairs, held-out prediction error on unseen layer pairs, compositional error between direct and composed transitions, and the relationship between these errors and generation quality. We report cross-timestep prediction only as a preliminary diagnostic because it requires additional target activations. These measurements do not prove a causal mechanism, but they test whether the empirical gains are consistent with the transition-learning interpretation.

\subsection{Skipped-Depth Prediction}
\label{sec:skip_prediction}

We evaluate skipped-depth prediction by replacing a block interval with the predicted hidden state and then resuming the remaining backbone. The primary measurements are output error against the full model, generation quality under skipped execution, and wall-clock speed. We treat this as an auxiliary capability rather than the central claim.

\subsection{Representation Analysis}
\label{sec:representation_analysis}

Finally, we test whether ReSync changes the backbone's intermediate representations in ways that are consistent with the proposed mechanism. The main motivating diagnostics are summarized in \Cref{fig:projection_mismatch_observation,fig:projection_layout_observation}: activation RMS or token norm measures statistical mismatch across layers; raw and projected CKA measure whether the predictor bridges representational mismatch; and DINO-derived pseudo-mask or boundary agreement tests whether projected states recover spatial partitions beyond feature-space similarity. We treat affinity-based similarity, cosine similarity, SVCCA, rank spectra, and full temporal heatmaps as supplementary diagnostics in \Cref{app:layer_statistics_projection}.
