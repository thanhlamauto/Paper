\section{Layer Statistics and Projection Motivation}
\label{app:layer_statistics_projection}

This appendix contains the diagnostic experiments used to evaluate the projection-as-transition motivation. These analyses are intentionally kept outside the main paper: the main text focuses on the method and generation results, while the appendix tests whether projection heads mitigate statistical mismatch and whether DiT activations contain layout-recoverable spatial information. We use the main paper diagnostics only as evidence for a hypothesis, not as proof that all layers explicitly encode the same semantic layout.

\subsection{Diagnostic Protocol}
\label{app:diagnostic_protocol}

Unless otherwise stated, all diagnostics should be computed on the same checkpoints, dataset split, resolution, VAE, and layer sampling scheme used for the main experiments in \Cref{sec:experiments}. The recommended comparison set is:
\begin{itemize}[leftmargin=*]
    \item baseline DiT/SiT backbone without auxiliary representation guidance;
    \item LayerSync-style direct cosine alignment;
    \item LayerSync with a small projection head;
    \item LayerSync with a large projection head;
    \item pair-specific projector trained for selected layer pairs;
    \item ReSync with a shared conditional transition predictor.
\end{itemize}
For each method, report the layer pairs, timestep ranges, projection-head architecture, loss weights, and whether an EMA target is used. This mirrors the appendix style of self-representation alignment work, where implementation details and sensitivity studies are separated from the main result tables.

The main paper uses a compact diagnostic set: activation RMS or token norm for statistical mismatch, raw-versus-projected CKA for representation stitching, and DINO-derived pseudo-mask or boundary agreement for spatial-layout recovery. The appendix reports additional checks, including mean and variance, cosine similarity, SVCCA, stable and effective rank, singular-value spectra, affinity-based similarity, and temporal CKA heatmaps.

\subsection{Layerwise Statistical Mismatch}
\label{app:layerwise_statistical_mismatch}

The goal of this diagnostic is to test whether direct alignment is difficult because different layers may occupy statistically different representation spaces. For each hidden state, compute RMS activation, token norm, channelwise variance, stable rank, effective rank, cosine similarity, SVCCA, and linear CKA across layer pairs. Results should be reported both by absolute layer index and by layer distance. If distant layer pairs show larger shifts in scale, variance, rank, or similarity structure, this would support the hypothesis that a projection or transition module can reduce feature-space mismatch before the alignment signal is applied to the backbone.

\subsection{Projection Capacity and Layer Gap}
\label{app:projection_capacity_layer_gap}

This ablation tests whether projection heads act as useful adapters or simply absorb the auxiliary loss. The key comparison is direct alignment versus small and large projection heads across near and far layer pairs. If projection heads mitigate mismatch, we expect a small projection head to improve generation quality and reduce alignment difficulty while still transmitting a meaningful gradient to the diffusion backbone. An overly expressive projection may reduce alignment loss without improving FID, which would suggest that it solves the auxiliary task locally rather than improving the model's internal representations.

\subsection{Layout-Recoverable Spatial Information}
\label{app:layout_recoverable_information}

Raw cosine similarity and CKA do not fully measure whether two activations preserve the same spatial layout. Affinity-based similarity to DINO features is also only a weak proxy, since it remains a second-order representation-similarity measure and can still be affected by feature anisotropy or rank structure. We therefore treat layout as a recoverable spatial object extracted from activation maps rather than as raw feature equality. The main probes should convert activations into spatial partitions or boundaries and compare them with external perceptual proxies, such as DINO-derived pseudo masks, image edges, or object masks when available. We will treat a layer as containing layout-recoverable spatial information only if projected or spatial-normalized activations improve pseudo-mask IoU, Boundary-F1, or region agreement relative to raw activations and pass same-image versus different-image or token-shuffle controls.

\subsection{Transition Generalization}
\label{app:transition_generalization}

This diagnostic distinguishes a shared depth-transition model from a collection of pair-specific projection heads. Train the transition predictor on a subset of layer pairs, then evaluate prediction and composition error on held-out layer pairs at the same denoising timestep. As a preliminary extension, also test cross-timestep prediction on a small model when additional target activations are available. The expected evidence for ReSync is not only lower pairwise prediction error, but also lower compositional error when a long depth transition is decomposed into shorter transitions. This would support the interpretation that ReSync learns reusable internal representation dynamics rather than a local adapter for a single target.
