\section{ReSync Algorithm and Compute}
\label{app:resync_algorithm_compute}

This appendix follows the same organizational principle as LayerSync: the main paper reports compact result tables, while the appendix separates implementation details, baseline protocol, sensitivity studies, metrics, and representation diagnostics. For ReSync, the split is adapted to the method's learned transition predictor and compositional loss. The relevant appendix groups are algorithm and compute, ImageNet and HumanML3D experimental details, transition-pair and timestep ablations, predictor-capacity controls, metric definitions, representation diagnostics, and qualitative examples. We omit LayerSync appendix topics that are not part of the current ReSync claims, such as audio generation, video generation, and post-hoc block dropping.

Algorithmically, ReSync adds a lightweight auxiliary branch during training. The backbone forward pass produces source and target activations, the transition predictor maps source activations to target-layer predictions, and the auxiliary objective combines prediction, composition, output-consistency, and weak layer-diversity losses. The predictor is removed for ordinary sampling, so inference cost is unchanged.

\subsection{Transition Pair and Predictor Design}
\label{app:transition_pair_predictor_design}

The choice of transition pairs plays the same practical role as layer selection in direct self-alignment, but ReSync uses one shared predictor rather than a separate head for each source--target pair. We therefore sample pairs from a bounded range of layer gaps. Very short gaps make the prediction task nearly trivial, while very long gaps can make the target representation too distant to provide stable supervision early in training. Unless otherwise stated, the main runs use a mixture of short and medium gaps; the gap distribution is ablated in \Cref{app:resync_ablation_protocols}.

We also avoid making the final decoder-like blocks the dominant targets. Prior analyses of diffusion representations show that semantic quality is heterogeneous across depth and that the last blocks become increasingly tied to output reconstruction~\citep{mukhopadhyay2024,kim2025,stracke2025,haghighi2026layersync}. We therefore treat the valid source range, target range, and gap distribution as hyperparameters. The predictor is kept lightweight and shared across pairs through layer and timestep conditioning, so the method tests whether a compact module can capture reusable internal transitions instead of memorizing pair-specific alignments.

When using source groups, define them by block index for a backbone with $L$ blocks, for example $\mathcal{S}_{\mathrm{early}}=\{1,\dots,\lfloor0.2L\rfloor\}$, $\mathcal{S}_{\mathrm{mid}}=\{\lfloor0.4L\rfloor,\dots,\lfloor0.6L\rfloor\}$, and $\mathcal{S}_{\mathrm{late}}=\{\lceil0.8L\rceil,\dots,L\}$. This mirrors the appendix style of self-representation alignment work, where implementation details and sensitivity studies are separated from the main result tables.

The main paper uses a compact diagnostic set: activation RMS or token norm for statistical mismatch, raw-versus-projected CKA for representation stitching, and generation metrics for direct versus projected self-alignment. The appendix reports additional checks, including mean and variance, cosine similarity, SVCCA, stable and effective rank, singular-value spectra, affinity-based similarity, spatial-layout probes, temporal CKA heatmaps, and gradient-alignment diagnostics.

\subsection{Metric and Measurement Definitions}
\label{app:metric_definitions}

Some diagnostics used in this paper are standard benchmark metrics with established citations, while others are simple descriptive measurements over hidden states. To avoid ambiguity for the latter case, we define the measurements used in the main text and appendix here. Let $X,Y\in\mathbb{R}^{M\times C}$ denote two activation matrices flattened over images and spatial tokens, where rows index tokens and columns index channels. Let $X_c$ and $Y_c$ denote column-centered activations.

\paragraph{Generation metrics.}
FID, IS, precision, and recall are reported using the standard image-generation evaluation protocol cited in the main text. FID compares Gaussian approximations of real and generated Inception features,
\begin{equation}
\operatorname{FID}
=
\|\mu_r-\mu_g\|_2^2
+
\operatorname{Tr}\left(
\Sigma_r+\Sigma_g
-2(\Sigma_r\Sigma_g)^{1/2}
\right),
\end{equation}
where lower is better. IS is $\exp(\mathbb{E}_x[\operatorname{KL}(p(y|x)\|p(y))])$, where higher indicates confident and diverse classifier predictions. Precision measures the fraction of generated samples lying inside the real-data feature manifold, while recall measures the fraction of real samples covered by the generated feature manifold. When sFID is reported, it uses the same Frechet form as FID but computes statistics from spatial feature maps rather than pooled image features.

\paragraph{Human motion metrics.}
For HumanML3D, FID uses the same Frechet form but computes the mean and covariance from motion-evaluator embeddings rather than image Inception embeddings. R-Precision measures text-motion retrieval accuracy: for each generated motion and its text prompt, the evaluator ranks the matching text among distractor texts by embedding similarity, and the score is the fraction of examples where the correct text appears in the top $R$ retrieved candidates. We report top-3 R-Precision unless otherwise stated.

\paragraph{Activation scale and spectrum.}
The activation RMS is $\sqrt{(MC)^{-1}\sum_{m,c}X_{m,c}^2}$. The token norm is $M^{-1}\sum_m\|X_{m,:}\|_2$; when comparing models with different channel counts, we report the normalized token norm $M^{-1}\sum_m\|X_{m,:}\|_2/\sqrt{C}$. Channelwise variance is the mean variance of each channel across the flattened token set. For spectrum diagnostics, we first z-normalize each channel,
\begin{equation}
Z_{m,c}
=
\frac{X_{m,c}-\mu_c}{\sigma_c+\varepsilon},
\qquad
\mu_c=\frac{1}{M}\sum_{m=1}^{M}X_{m,c},
\qquad
\sigma_c^2=\frac{1}{M}\sum_{m=1}^{M}(X_{m,c}-\mu_c)^2 .
\end{equation}
If $s_1\geq \cdots \geq s_r$ are singular values of $Z$, stable rank is
\begin{equation}
\operatorname{srank}(Z)
=
\frac{\|Z\|_F^2}{\|Z\|_2^2}
=
\frac{\sum_i s_i^2}{s_1^2},
\end{equation}
and effective rank is $\exp(-\sum_i p_i\log p_i)$ with $p_i=s_i/\sum_j s_j$. The z-normalization removes per-channel scale shifts, so rank diagnostics reflect spectral concentration of the representation rather than raw activation magnitude. Higher rank indicates less spectral concentration.

\paragraph{Representation similarity.}
Tokenwise cosine similarity is
\begin{equation}
\operatorname{cos}(X,Y)
=
\frac{1}{M}
\sum_{m=1}^M
\frac{X_{m,:}^{\top}Y_{m,:}}
{\|X_{m,:}\|_2\|Y_{m,:}\|_2+\varepsilon},
\end{equation}
and tokenwise cosine distance is $1-\operatorname{cos}(X,Y)$. Linear CKA is
\begin{equation}
\operatorname{CKA}(X,Y)
=
\frac{\|X_c^{\top}Y_c\|_F^2}
{\|X_c^{\top}X_c\|_F\|Y_c^{\top}Y_c\|_F+\varepsilon},
\end{equation}
following the representation-similarity convention of \citet{kornblith2019similarity}; related transformer-representation analyses also use CKA-style diagnostics~\citep{raghu2021visiontransformers}. SVCCA is reported as the mean canonical correlation after projecting $X_c$ and $Y_c$ onto their leading singular subspaces; the retained variance threshold or retained component count should be stated with each result. Affinity error compares pairwise token relations rather than raw features: after row-normalizing activations, form $S_X=\tilde X\tilde X^\top$ and $S_Y=\tilde Y\tilde Y^\top$, then report $\|S_X-S_Y\|_F/M$ or its centered correlation equivalent. Lower affinity error or higher affinity correlation indicates more similar token geometry. Temporal CKA uses the same definition while holding the layer fixed and varying the denoising timestep, for example $\operatorname{CKA}(A_\ell^t,A_\ell^s)$. We report it as a heatmap over $(t,s)$ for selected layers and as a local-neighborhood average for nearby timesteps. This diagnostic supports the broader transition-prediction view but is not part of the main same-timestep ReSync objective.

\paragraph{Gradient-alignment diagnostics.}
To test whether projection reduces conflict with the generative objective, compute
\begin{equation}
\operatorname{gradcos}
=
\frac{
\left\langle
\nabla_\theta\mathcal{L}_{\mathrm{FM}},
\nabla_\theta\mathcal{L}_{\mathrm{aux}}
\right\rangle
}{
\|\nabla_\theta\mathcal{L}_{\mathrm{FM}}\|_2
\|\nabla_\theta\mathcal{L}_{\mathrm{aux}}\|_2
+\varepsilon
},
\end{equation}
where $\mathcal{L}_{\mathrm{aux}}$ is direct alignment, projected alignment, or ReSync. The parameter subset $\theta$ should be stated explicitly, such as source blocks only, all transformer blocks, or shared embeddings. Report the mean and confidence interval over matched batches, timesteps, layer pairs, and checkpoints; higher values indicate less conflicting auxiliary gradients.

\paragraph{Spatial-layout probes.}
Pseudo-mask IoU compares a mask or partition recovered from model activations with a reference partition, such as a DINO-derived pseudo mask. For binary masks, $\operatorname{IoU}(P,Q)=|P\cap Q|/|P\cup Q|$; for multiple regions, we report mean best-matched IoU and state the matching rule. Boundary-F1 compares predicted and reference boundary pixels with a fixed tolerance radius $\tau$: precision is the fraction of predicted boundary pixels matched to a reference boundary, recall is the fraction of reference boundary pixels matched by a prediction, and $\operatorname{Boundary\mbox{-}F1}=2PR/(P+R+\varepsilon)$. Same-image, different-image, and token-shuffle controls use the same metric but replace the reference or spatial ordering to test whether the score reflects image-specific layout rather than feature anisotropy.

\paragraph{Transition diagnostics.}
For a transition predictor $\Phi_{a\rightarrow b}$, prediction error is
\begin{equation}
E_{\mathrm{pred}}(a,b)
=
\mathbb{E}_{x,t}
\left[
d\left(\Phi_{a\rightarrow b}(A_a^t,t),\operatorname{sg}(A_b^t)\right)
\right],
\end{equation}
where $d$ is the tokenwise cosine distance unless otherwise stated. Held-out layer error uses the same formula on layer pairs excluded from predictor training. For $a<b<c$, compositional error compares a direct long transition with two composed short transitions:
\begin{equation}
E_{\mathrm{comp}}(a,b,c)
=
\mathbb{E}_{x,t}
\left[
d\left(
\Phi_{a\rightarrow c}(A_a^t,t),
\Phi_{b\rightarrow c}(\Phi_{a\rightarrow b}(A_a^t,t),t)
\right)
\right].
\end{equation}
Skipped-depth output error is measured against the full backbone output using normalized MSE, $\|\hat o_{\mathrm{skip}}-\hat o_{\mathrm{full}}\|_2^2/(\|\hat o_{\mathrm{full}}\|_2^2+\varepsilon)$, or the same tokenwise cosine distance when the output parameterization is normalized. Error--FID correlation is computed across matched runs or checkpoints using Pearson correlation by default; Spearman correlation can be reported as a rank-based robustness check.

\paragraph{Reporting rule for uncited diagnostics.}
Whenever a metric is not a named benchmark with an explicit citation, the table or caption should state the normalization axes, sample count, timestep range, layer-pair sampling rule, whether features are centered or normalized, and whether lower or higher is better. This keeps descriptive diagnostics interpretable without treating them as standalone benchmark claims.

\subsection{Residual Transition Identity}
\label{app:residual_transition_derivation}

For completeness, we state the residual-transition identity for a DiT/SiT-style transformer stack~\citep{vaswani2017attention,peebles2023scalable,ma2024sit}. The purpose is only to clarify the transition notation used in the appendix diagnostics. It is not an additional architectural assumption.

\begin{figure}[!htbp]
    \centering
    \fbox{%
    \begin{minipage}[c][0.24\textheight][c]{0.9\linewidth}
        \centering
        \textbf{[PLACEHOLDER: DiT residual block and telescoping transition]}\\[0.5em]
        Left: one DiT block with AdaLN, MSA, gated residual, AdaLN, MLP, gated residual.\\
        Right: stacked blocks from $a$ to $b$ showing
        $A_b^t=A_a^t+\sum_{i=a}^{b-1}\Delta_i^t$.
    \end{minipage}}
    \caption{\textbf{Residual view of a DiT stack.} A DiT block updates its token activation through attention and MLP residual branches modulated by timestep/class conditioning. Stacking such blocks yields a telescoping decomposition of a later activation into the source activation plus accumulated residual updates.}
    \label{fig:appendix_dit_residual_stack}
\end{figure}

Let $A_i^t$ be the token activation entering block $i$ at denoising timestep $t$, and let $c$ denote the conditioning signal. A standard DiT block applies adaptive normalization, self-attention, and an MLP inside two residual branches. We write it as
\begin{equation}
\bar A_i^t
=
\operatorname{AdaLN}^{\mathrm{msa}}_i(A_i^t;t,c),
\qquad
U_i^t
=
A_i^t
+
g_i^{\mathrm{msa}}(t,c)\odot
\operatorname{MSA}_i(\bar A_i^t),
\label{eq:appendix_dit_attn_branch}
\end{equation}
\begin{equation}
\bar U_i^t
=
\operatorname{AdaLN}^{\mathrm{mlp}}_i(U_i^t;t,c),
\qquad
A_{i+1}^t
=
U_i^t
+
g_i^{\mathrm{mlp}}(t,c)\odot
\operatorname{MLP}_i(\bar U_i^t),
\label{eq:appendix_dit_mlp_branch}
\end{equation}
where the gates $g_i^{\mathrm{msa}}$ and $g_i^{\mathrm{mlp}}$ are broadcast over tokens, as in DiT-style adaptive modulation. The exact implementation of AdaLN or gating is not important for the identity; the key property is the residual addition around each branch. Combining the two branches gives the block update
\begin{equation}
\Delta_i^t
\equiv
A_{i+1}^t-A_i^t
=
g_i^{\mathrm{msa}}(t,c)\odot
\operatorname{MSA}_i(\bar A_i^t)
+
g_i^{\mathrm{mlp}}(t,c)\odot
\operatorname{MLP}_i(\bar U_i^t).
\label{eq:appendix_dit_delta}
\end{equation}
For any two layers $a<b$, the depth transition is obtained by summing these residual increments:
\begin{equation}
\begin{aligned}
A_b^t
&=
A_a^t
+
\sum_{i=a}^{b-1}\left(A_{i+1}^t-A_i^t\right) \\
&=
A_a^t
+
\sum_{i=a}^{b-1}\Delta_i^t .
\end{aligned}
\label{eq:appendix_telescoping}
\end{equation}
This telescoping decomposition shows that a later activation equals an earlier activation plus the computation accumulated by intervening blocks. The identity is purely algebraic and does not by itself prove that the accumulated update is easy to predict or semantically meaningful.

\subsection{Layerwise Statistical Mismatch}
\label{app:layerwise_statistical_mismatch}

The goal of this diagnostic is to test whether direct alignment is difficult because different layers may occupy statistically different representation spaces. For each hidden state, compute RMS activation, token norm, channelwise variance, stable rank, effective rank, cosine similarity, SVCCA, and linear CKA across layer pairs. Results should be reported both by absolute layer index and by layer distance. If distant layer pairs show larger shifts in scale, variance, rank, or similarity structure, this would support the hypothesis that a projection or transition module can reduce feature-space mismatch before the alignment signal is applied to the backbone.

\subsection{Layout-Recoverable Spatial Information}
\label{app:layout_recoverable_information}

Raw cosine similarity and CKA do not fully measure whether two activations preserve the same spatial layout. Affinity-based similarity to DINO features is also only a weak proxy, since it remains a second-order representation-similarity measure and can still be affected by feature anisotropy or rank structure. We therefore treat layout as a recoverable spatial object extracted from activation maps rather than as raw feature equality. The main probes should convert activations into spatial partitions or boundaries and compare them with external perceptual proxies, such as DINO-derived pseudo masks, image edges, or object masks when available. We will treat a layer as containing layout-recoverable spatial information only if projected or spatial-normalized activations improve pseudo-mask IoU, Boundary-F1, or region agreement relative to raw activations and pass same-image versus different-image or token-shuffle controls.

\subsection{Transition Generalization}
\label{app:transition_generalization}

This diagnostic distinguishes a shared depth-transition model from a collection of pair-specific projection heads. Train the transition predictor on a subset of layer pairs, then evaluate prediction and composition error on held-out layer pairs at the same denoising timestep. The expected evidence for ReSync is not only lower pairwise prediction error, but also lower compositional error when a long depth transition is decomposed into shorter transitions. This would support the interpretation that ReSync learns reusable internal representation dynamics rather than a local adapter for a single target.
